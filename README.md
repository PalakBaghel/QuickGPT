Tech Stack: React.js, Node.js, Express.js, MongoDB, OpenAI React SDK, Google Gemini API, ImageKit, Stripe (Payment Gateway)

Designed and developed a full-stack AI Chatbot application similar to ChatGPT and Google Gemini using the MERN stack.

Implemented secure user authentication (JWT-based signup/login) with protected routes and user session management.

Integrated Google Gemini AI model via OpenAI React SDK to generate AI-powered conversational text responses.

Built AI image generation functionality using prompt-based image creation with ImageKit integration.

Designed a credit-based usage system, allowing users to purchase credits through a secure online payment gateway (Stripe).

Implemented backend APIs for chat history management, credit tracking, and transaction handling.

Ensured optimized performance with efficient state management and RESTful API architecture.

Built a responsive and modern UI with React, improving user engagement and experience.

Deployed the full-stack application to production with environment-based configuration and secure API handling.
